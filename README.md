# Advent of Code 2022

My solutions to the [advent of code](https://adventofcode.com) challenge - 2022

## Installation

```bash
python3.10 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Running

```bash
python <day>/solution.py
```

## Testing

```bash
pytest [day]
```

## Linting

```bash
mypy [.|<day>]
```
